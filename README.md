## Features

- 🚇 **Metro Route Planning** - Find the best routes between stations
- 🌐 **Bilingual** - Full support for Persian (RTL) and English (LTR)
- 📱 **PWA** - Install on any device, works offline
- 🌙 **Dark/Light Theme** - Automatic theme switching
- ⚡ **Fast** - Built with Next.js App Router & Turbopack

## ویژگی‌ها

- 🚇 **مسیریابی مترو** - یافتن بهترین مسیر بین ایستگاه‌ها
- 🌐 **دوزبانه** - پشتیبانی فارسی و انگلیسی
- 📱 **نصب روی گوشی** - کار می‌کند بدون اینترنت
- 🌙 **تم تاریک/روشن** - تعویض خودکار تم
- ⚡ **سریع** - ساخته شده با Next.js

```bash
# Start dev server
pnpm dev
```
