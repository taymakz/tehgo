
# TehGo - Tehran Metro Guide

**Tehran Metro Route Planner**

🌐 https://tehgo.ir

---

## Features

- 🚇 **Metro Route Planning** – Find the fastest and best routes between metro stations
- 🌐 **Bilingual Support** – Fully supports English (LTR) and Persian (RTL)
- 📱 **PWA (Progressive Web App)** – Install on any device and use offline
- 🌙 **Dark / Light Mode** – Automatic theme switching based on system preference
- ⚡ **High Performance** – Built with Next.js App Router and Turbopack

---

## Tech Stack

| Category        | Technology              |
|----------------|------------------------|
| Framework       | Next.js (App Router)   |
| Language        | TypeScript             |
| Styling         | Tailwind CSS           |
| UI Components   | Radix UI, shadcn/ui    |
| State Management| Zustand, TanStack Query |
| PWA             | Service Worker, Web Manifest |

---

## Quick Start

```bash
# Install Pnpm
npm i -g pnpm@latest

# Install dependencies
pnpm install

# Start development server
pnpm dev
````

Open: [http://localhost:3000](http://localhost:3000)
